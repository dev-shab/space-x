# Steps we did for this project

- Use Vite to create the React app
- Install Material UI and Tailwind CSS
- Implement the Header Component
- Configure Redux store data
- Create Slice for storing launchesdata
- Fetch all launches from the API - customHook
- Fetch all launchpads from the API to get the location of launches - customHook
- Fetch all payloads from the API to get the orbit info - customHook
- Fetch all rockets from the API to get the rocket info - customHook
- Utilise the Material UI Table to display the data in a tabular format
